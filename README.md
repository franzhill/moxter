
A lightweight, YAML-driven testing framework for Spring APIs leveraging MockMvc designed to orchestrate JUnit test scenarios, Moxter sits in the sweet spot between fast yet narrow-scoped unit testing with Mockito and comprehensive yet infrastructure-heavy external API testing with tools like Postman.
