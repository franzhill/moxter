



Moxter is a lightweight, YAML-driven testing framework for Spring APIs leveraging MockMvc designed to orchestrate JUnit test scenarios, 

sits in the sweet spot between fast yet narrow-scoped unit testing with Mockito and comprehensive yet infrastructure-heavy external API testing with tools like Postman.



With just a simple test like this one, we already have saved ourselves tedious boilerplate code, while conveying clarity, separation of concern and reusability.
